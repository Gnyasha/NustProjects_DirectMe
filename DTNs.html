DHCP Operation (DORA Process)

+ 
1. 📡 Discover → Broadcast (UDP 67)
2. 📦 Offer    → Server → Client (UDP 68)
3. 📨 Request  → Client → Server
4. ✅ Ack      → Final confirmation

Allocation Methods:

+---------------------+---------------------------+
| Dynamic             | Temporary leases (common) |
| Automatic           | Permanent auto-assignment |
| Manual              | Reserved MAC-IP mapping   |
+---------------------+---------------------------+

Subnetting Mastery

+

Basic Formula:

Number of Subnets = 2^(network bits borrowed)
Hosts per Subnet = 2^(host bits remaining) - 2

CIDR Example:

192.168.1.0/24 → 256 hosts
/26 → 4 subnets (64 hosts each)
Subnets: 192.168.1.0-63, 64-127, 128-191, 192-255

Expert Networking Tips

+ 
🔧 Subnetting Pro Tip:
Always reserve first/last addresses → Network & Broadcast

🌐 DHCP Best Practice:
Set lease times ↓ for mobile devices, ↑ for servers

🛡️ Security Essential:
Use DHCP Snooping to prevent rogue servers

🔍 Troubleshooting Flow:
Physical → IP Config → DNS → Firewall
Final Wisdom: Master binary/decimal conversion for subnetting, and always document your IP allocations!
